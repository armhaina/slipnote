# Slipnote

<p align="center"><a href="https://symfony.com" target="_blank">

</a></p>

Slipnote — это небольшой пет-проект для работы с заметками, который позволяет создавать, редактировать и организовывать
заметки с возможностью полнотекстового поиска. Проект включает REST API на Symfony с JWT-аутентификацией. Реализованы
тесты, валидация и обработка ошибок для отказоустойчивой работы.

Тесты
------------

1. Запустите команду: `make test-init`
2. Запустите команду: `make test-run`
